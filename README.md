## Project Overview
This project is an **Employee Attendance Management System** designed using **MySQL** to manage employee attendance records in an organization.  
It demonstrates the use of **relational database concepts** such as table creation, constraints, relationships, and SQL queries.

The database stores information about **employees, departments, attendance records, and leave requests** in a structured and organized manner.

---

## Objectives
- To design a relational database for employee attendance management  
- To practice core SQL concepts using MySQL  
- To implement relationships using primary and foreign keys  
- To track employee attendance and leave records using SQL queries  

---

## Technologies Used
- **Database:** MySQL  
- **Tool:** MySQL Workbench  
- **Language:** SQL
